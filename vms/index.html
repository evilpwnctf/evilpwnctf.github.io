Common Issues

Common Issues
1. If you get no IP address, probably you're using another virtualization software (E.g. Oracle VirtualBox)
2. If you get error "SHA1 digest of file xxxx.ovf does not match manifest" delete the .mf file in the folder




FAQ (Frequently Asked Question)

FAQ (Frequently Asked Question)
1. Q : What's the Flag Format? | A : Like the challenges EVILPWNCTF{sha256_string}
2. Q : Where can i find the flag? | A : There is only 1 flag under /root/root.txt
3. Q : How to submit a flag? | A : Submit the flag by DM @EvilPWN CTF BOT : $submit EVILPWNCTF{sha256_string}
4. Q : How to find machine's IP address? | A : IP address already exists when you boot the box, but you can use arp-scan utility too : arp-scan --localnet
